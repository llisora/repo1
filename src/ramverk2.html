Tailwind

Länk till webbplatsen

Ett större ramverk som innehåller många olika klasser med olika användningsområden.

Tailwind använder framför allt olika så kallade "utility classes".

T.ex "bg-blue" är klassen för en blå bakgrundsfärg. Ett ramverk som kan användas till mycket

och kan användas till att style:a det mesta på en webbplats.

Fördelar:

Relativt enkelt att installera. Enkelt att använda.

Många olika klasser och lätt att läsa sig till information i dokumentationen.

Nackdelar:

Koden blir väldigt plottrig när det ska användas så många olika klasser.

För någon som kan CSS känns detta ibland lite omständigt och långsammare än vanlig CSS.
