Publicering

Jag har valt att använda Netlify för att publicera min webbplats.

Netlify är ett smidigt verktyg som kopplar upp ens webbplats till GitHub och de repository man använder.

Detta betyder att när ändringar sker i ens repository så sker det även på webbplatsen!


Netlify finns både som gratis att använda eller olika betalversioner. I gratisversionen så kan man endast

vara en person per "team" (dvs du kan ej jobba flera på ett projekt) och du får endast 1000GB "traffic" per månad.

Flera andra avancerade funktioner utelämnas även i gratisversionen men det funkar utmärkt till uppgifter som dessa.


Netlify har stöd för flera olika programmeringsspråk så som JavaScript och TypeScript, men har inte stöd för PHP och liknande.

Att ladda upp hemsidan och få den att fungera i sig var ganska enkelt. Man behöver ha viss kunskap om GitHub och liknande för att förstå hur man

ska göra, eftersom att man använder sitt GitHub-repo för att få upp filerna på Netlify.


Annars tycker jag att det har varit smidigt med Netlify och väldigt bra att man ser ändringarna (typ) direkt.

Värt att notera är att man även kan välja att ladda upp filerna direkt på Netlify om man inte känner sig bekväm med GitHub.
